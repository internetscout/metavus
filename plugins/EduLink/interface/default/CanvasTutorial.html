Embedding a resource in Canvas

Resources can be embedded in Canvas in a number of ways. The two most common are as Modules and as Assignments.

  • Select a course to edit and navigate to the front page for that course.
  • Embedding in a Module
    1. From the course navigation at left, select Modules.
      gUIFileTag("Canvas_Tutorial_M1.jpg") ?>
    2. If this resource should be part of a new module, use the + Module button at top right to create the new module.
      gUIFileTag("Canvas_Tutorial_M2.jpg") ?>
    3. Press the + button in the header of the module where this resource should live.
      gUIFileTag("Canvas_Tutorial_M3.jpg") ?>
    4. Select External Tool from the Add menu.
      gUIFileTag("Canvas_Tutorial_M4.jpg") ?>
    5. Select getConfigSetting("ServiceName") ?> from the list of available tools.
    6. Follow the instructions in the Locating a Resource section below, proceeding with the next step in this section afterwards.
    7. Press the Add Item button.
      gUIFileTag("Canvas_Tutorial_M5.jpg") ?>
    8. Your selected resource will now appear as a component of a Canvas module. It can be published using the 'Publishe' button at right of the content. This button appears as a 'no' icon for unpublished entries and a green checkmark for published entries. Alternately, you may leave the content unpublished for reference materials that should not be visible to students.
      gUIFileTag("Canvas_Tutorial_M6.jpg") ?>
  • Embedding in an Assignment
    1. From the course navigation at left, select Assignments.
      gUIFileTag("Canvas_Tutorial_A1.jpg") ?>
    2. Click the + Assignment button at top right.
      gUIFileTag("Canvas_Tutorial_A2.jpg") ?>
    3. Under Submission type, select External Tool, then press Find
      gUIFileTag("Canvas_Tutorial_A3.jpg") ?>
    4. Press the Find button
      gUIFileTag("Canvas_Tutorial_A4.jpg") ?>
    5. Select getConfigSetting("ServiceName") ?> from the list.
    6. Follow the instructions in the Locating a Resource section below, proceeding with the next step in this section afterwards.
    7. Press the Select button in Canvas.
      gUIFileTag("Canvas_Tutorial_A5.jpg") ?>
    8. Give your assignment a title. You can also provide introductory text that will be displayed above the resource.
      gUIFileTag("Canvas_Tutorial_A6.jpg") ?>
    9. Press Save & Publish
      gUIFileTag("Canvas_Tutorial_A7.jpg") ?>
    10. Your selected resource will now appear embedded in a Canvas assignment.
      gUIFileTag("Canvas_Tutorial_A8.jpg") ?>
  • Locating a Resource
    1. Use the search box and/or the facets shown at right to locate the resource(s) you want.
    2. Use the Select buttons to make your resource selections.
      gUIFileTag("Canvas_Tutorial_L1.jpg") ?>
    3. When you've selected all the resources you want, press the Send to LMS button to forward your selections back to Canvas.
      gUIFileTag("Canvas_Tutorial_L2.jpg") ?>
