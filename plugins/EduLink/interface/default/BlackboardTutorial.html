Embedding a resource in Blackboard

  1. Select a course to edit and navigate to the front page for that course.
  2. Select Content from the menu at left.
    gUIFileTag("Blackboard_Tutorial_1.jpg") ?>
  3. Select getConfigSetting("ServiceName") ?> from the Build Content drop down menu.
    gUIFileTag("Blackboard_Tutorial_2.jpg") ?>
  4. Use the search box and/or the facets shown at right to locate the resource(s) you want.
  5. Use the Select buttons to make your resource selections.
    gUIFileTag("Blackboard_Tutorial_3.jpg") ?>
  6. When you've selected all the resources you want, press the Send to LMS button to forward your selections back to Canvas.
    gUIFileTag("Blackboard_Tutorial_4.jpg") ?>
  7. Your selected resource(s) will now appear in the course content list.
    gUIFileTag("Blackboard_Tutorial_5.jpg") ?>
